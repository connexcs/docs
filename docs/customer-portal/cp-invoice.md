## Overview

The **Invoice module** provides a comprehensive and user-friendly interface for managing billing records, enabling users to efficiently view, filter, and download invoices while maintaining full visibility into account charges.

---

## Invoice List View

The main screen displays a **tabular list of invoices** with the following columns:

| Column| Description|
| ------|------------|
| **Invoice ID**   | Unique identifier for each invoice |
| **Name** | Account or customer name associated with the invoice |
| **Invoice Date** | Date when the invoice was generated |
| **Total** | Total invoice amount (e.g., USD)|
| **Paid**| Payment status (Paid / Unpaid / Pending)|
| **Action** | Available actions for each invoice |

### Available Actions

Each invoice provides the following actions:

* **View Invoice**
  Opens the invoice in a web view for quick review.

* **Download as PDF**
  Downloads a copy of the invoice for offline access, sharing, or record-keeping.

---

## Search & Filtering

The invoice table includes tools to quickly locate specific invoices:

* **Search Bar (Top Right)**
  Allows keyword-based search (e.g., Invoice ID, Name).

* **Column Filters**
  Each column supports filtering to refine results.

* **Sorting**
  Columns can be sorted to organize data (e.g., by date or amount).

---

## Column Customization (Pivot Mode)

The portal provides a **Pivot Mode / Column Selector** panel:

* Toggle columns on/off using checkboxes
* Customize visible data based on user preference
* Rearrange columns for better analysis

---

## Typical Use Cases

* Reviewing billing history
* Downloading invoices for accounting
* Tracking unpaid invoices
* Verifying charges and usage
