## Overview

The **Interconnect** section provides a **read-only view** of the **authorised IP addresses**, and **RTP servers** used for call routing and media handling on your account.

**It shows**:

- Where outbound calls are sent (**Termination**)

- Which IP addresses are authorised to send inbound calls (**Origination**)

- Which **RTP servers** handle call media

---

### RTP Servers

* Handles **media (audio) flow** for calls
* Ensures proper voice transmission between endpoints
